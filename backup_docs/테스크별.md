\# \[FE-01\] AppShell 골격(Topbar \+ Sidebar \+ Main)

\#\# 🎯 목적  
좌측 GNB \+ 상단 Topbar 고정 레이아웃 구축

\#\# 📦 범위  
\- Topbar(로고, 글로벌 검색, 알림, 도움말, 프로필)  
\- Sidebar(GNB: Home/Leads/Tickets/Appointments/Dashboards/Channels/Profiles/Settings/Audit)

1\. Topbar/Sidebar sticky 동작, 콘텐츠만 스크롤  
2\. 현재 라우트와 GNB 활성화 동기화(aria-current)  
3\. 기본 접근성 속성(aria-label, tab 포커스) 적용

\#\# 🔍 QA 체크리스트  
\- \[ \] 1280px/1440px에서 레이아웃 깨짐 없음  
\- \[ \] 키보드로 Topbar 아이콘/메뉴 탐색 가능  
\- \[ \] GNB 링크 포커스 링 표시  

# [FE-02] 반응형 GNB & 모바일 드로어

## 🎯 목적
데스크톱 고정, 태블릿 접힘(collapsed), 모바일 드로어 구현

## 📦 범위
- 3단 브레이크포인트(≥1280/768–1279/<768)
- GNB 툴팁, 버거 버튼

## ✅ 완료조건 (AC)
1. 768–1279px: GNB 72px 아이콘 모드 + 툴팁
2. <768px: GNB 오버레이 드로어(ESC/백드롭/스와이프 닫힘)
3. 스크린리더에서 드로어 `aria-modal="true"`

## 🔍 QA 체크리스트
- [ ] 포커스 트랩/ESC 동작
- [ ] 리사이즈 시 상태 보존
