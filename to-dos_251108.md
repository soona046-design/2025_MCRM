  CREATE TABLE raw_metrics (
    id UUID PRIMARY KEY,

    synced_at TIMESTAMP,
    created_at TIMESTAMP,
    updated_at TIMESTAMP,

  );

### 3️⃣ API 연동 서비스 개선

#### 3.1 RawMetricService 생성
- [ ] **Upsert 로직 구현**
  ```php
  class RawMetricService {
    public function upsertMetrics($platform, $data) {
      foreach ($data as $metric) {
        RawMetric::updateOrCreate(
          [
            'platform' => $platform,
            'campaign_code' => $metric['campaign_code'],
            'period_label' => $metric['period_label'],
          ],
          [
            'impressions' => $metric['impressions'],
            'clicks' => $metric['clicks'],
            'cost' => $metric['cost'],
            'raw_json' => $metric,
            'synced_at' => now(),
          ]
        );
      }
    }
  }
  ```

- [ ] **버전 백업 로직**
  ```php
  public function backupVersion($rawMetricId) {
    $raw = RawMetric::find($rawMetricId);
    VersionedRawMetric::create([
      'raw_metric_id' => $rawMetricId,
      'version_no' => $this->getNextVersion($rawMetricId),
      'snapshot_data' => $raw->toArray(),
    ]);
  }
  ```
